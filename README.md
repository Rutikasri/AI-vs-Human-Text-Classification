# AI vs Human Text Classification

## Overview
This project classifies whether a given text is AI-generated or human-written using machine learning and NLP techniques.

## Features
- Text preprocessing (stopwords removal, lemmatization)
- Feature extraction using TF-IDF
- Classification using machine learning models
- Simple UI built with Streamlit

## Tech Stack
- Python
- Scikit-learn
- NLTK
- Streamlit

## Model Details
- Algorithms: Logistic Regression / Naive Bayes
- Vectorization: TF-IDF
